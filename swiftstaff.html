SwiftStaff

Description

SwiftStaff is an app for iOS and Android to help restaurants and cafes find temporary workers. Our app consists of a React Native front-end with Kotlin based REST API back-end and a MongoDB database.

Swiftstaff was developed in an iterative process based on user stories and feedback. Features were developed utilising feedback from real restaurant owners
and managers as well as temporary workers, improving the experience of the app and tailoring it to their needs.

Uses Google's Firebase API and Apple notification service to give real time job notifications, Google maps API to show location of jobs, and a two way rating system for employees and employers. Also uses websockets for live job updates.

Created as part of Designing for Real People course at Imperial College London, with Terence Hernandez, Raghav Khanna and Suchismata Paul. May be open sourced soon.
